# Ejecutar en development mode

1. Clone repo
2. Ejecutar
```
npm i
```
3. Nest CLI (instalado)
```
npm i -g @nestjs/cli
```

4. Levantar DB
```
docker-compose up -d
```

## STACK
* Nest framework
* Mongodb
